# ♻️ Scrapify

**Scrapify** is a dedicated web platform designed to promote sustainability through waste donation. It provides a seamless interface for individuals and organizations to donate their recyclable waste to those who can process it, fostering a circular economy.

## 🚀 Features

- **Waste Donation Portal**: Easy submission of waste details for donation.
- **Donor & Recycler Profiles**: Personalized accounts for tracking donations and contributions.
- **Impact Dashboard**: Visual representation of waste recycled and environmental impact.
- **Real-time Notifications**: Updates on donation status and collection.

## 🛠️ Tech Stack

- **Backend**: PHP, MySQL
- **Frontend**: HTML5, CSS3, JavaScript
- **Frameworks**: Hack (for enhanced PHP features)

## 📂 Directory Structure

- `/html`: Frontend structural files.
- `/css`: Styling and layout resources.
- `/js`: Interactive logic for the platform.
- `/api`: Backend API endpoints for donation processing.

## 🚀 Getting Started

1. Clone the repository.
2. Import the database schema from the `/database` folder.
3. Configure your local PHP server and point it to the project root.
